# Maya Tech Exam by Phillip Cui

Hi! This an app made based on the technical specs provided for the exam

## Cloning the repository

git url:  https://github.com/pelep-cui/MayaAndroidExam.git

You can use the git clone command to clone the repository. Here is the command:

git clone https://github.com/pelep-cui/MayaAndroidExam.git

You can also use any GUI git clients like Sourcetree by copying the git url link and pasting the url on the clone project dialog.

## Importing the project

In Android Studio, go to File > Open > Go to the directory where you clones the project > Select the folder.

Just wait for Android Studio to complete importing the project.

## Running the app

In the top part of the application you will find the play button. Click that button to start the building process. After the project finished building it will then automatically launch the app on the device selected. If a device it connected it will automatically launch the app on that device.

## How to run the unit tests

In the top-left side panel you will see the project files. Go to  app > src, find the ‘test’ folder. Right click on it, a popup will show up then select run Test in ‘MayaExamApplication’.
